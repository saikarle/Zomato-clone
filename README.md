# 🍽️ Zomato Clone

A frontend clone of the Zomato landing page built using **HTML, CSS, and JavaScript**. This project recreates the modern look and feel of Zomato's homepage while helping me improve my frontend development skills through hands-on practice.

---

## 🚀 Features

- 🧭 Navigation Bar
- 🎯 Hero Section
- 🔍 Search Bar
- 🔐 Login Page
- 📝 Sign Up Page
- 🍴 Add Restaurant Page
- 💼 Investor Relations Page
- 📄 Footer
- ✨ Smooth Animations
- 🎨 Interactive Buttons

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## ▶️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/saikarle/zomato-clone.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
